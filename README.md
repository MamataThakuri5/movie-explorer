
# 🎬 Movie Explorer

A simple web app to search movies, view details, and save your favorites — built with vanilla HTML, CSS and JavaScript using the OMDb API.

## Features
- 🔍 Live search (no button needed — just type!)
- 🎞️ Click any movie to see full details (plot, rating, poster)
- ❤️ Save movies to favorites
- 💾 Favorites persist after refresh (localStorage)
- ⏳ Debounced search to avoid excessive API calls

## Technologies used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Fetch API, Promises, localStorage, debounce)
- OMDb API

## How to run locally
1. Clone this repo
2. Get a free API key from omdbapi.com
3. Add your key in script.js
4. Open index.html in your browser

## What I learned
This project helped me practice working with real APIs, debouncing user input, and managing state with localStorage — concepts I learned from Namaste JavaScript.
